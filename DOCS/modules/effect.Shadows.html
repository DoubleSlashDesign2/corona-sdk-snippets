Shadow:AddProperty (name, curve, v0, dv)
Fits a shadow property to a curve.

The final value of the property at a time t will be v0 + scale * dv.

Parameters:

  • name string Property name, which should correspond to some numeric property of the shadow (typically, this will be a display object property).
  • curve callable

    Curve function, called as

    scale = curve(t)
  • v0 number Original value of property.
  • dv number Value delta.

See also:

    Shadow:ResolveProperties
Shadow:AddMultipleProperties (curve, args)
Convenience method to add several properties using the same curve.

Parameters:

  • curve callable Curve function.

    In case of side effects, note that this function is called once per property.

  • args table Key-value pairs, with each key corresponding to name and each value a table of the form { v0 = v0, dv = dv }, cf. Shadow:AddProperty.
Shadow:ResolveProperties (t)
Resolves a shadow's properties at a given time.

Parameters:

  • t number A time, passed as input to each property's curve function.

See also:

  • Shadow:AddProperty
  • Shadow:AddMultipleProperties
Shadow (func, arg)
Creates a shadow effect, along with some useful state.

Parameters:

  • func callable Shadow function, called as

    result = func(shadow, arg[, true])

    This function is called on creation (this is when it receives the true argument), and on each update; shadow is the shadow object returned by this call.

    If result is ever "quit", the object is removed and the effect cancelled.

    If result was "report_quit" on the initial call, then after quitting the call func("quitting", arg) is performed, allowing e.g. for external cleanup.

  • arg Argument to func.

Returns:

    DisplayObject Shadow object. It can be removed to cancel the effect.
