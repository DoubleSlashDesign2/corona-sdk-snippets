BoolVars:GetBits_Array (arr)
Looks up several bool variables and sets each bit of an integer: 1 if the bool is true and 0 otherwise. The first variable is at bit 0.

Any leftover bits are set to 0.

Parameters:

  • arr array Array of bool variable names (up to 32).
