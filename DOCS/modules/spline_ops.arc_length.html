Module spline_ops.arc_length

Some utilities for calculating arc lengths.

For purposes of this module, an instance of type Vector is a value, e.g. a table, that has and / or receives number members x and y.

Lookup (add_01_wrapper[, lut])
Converts an arc length into a curve parameter, given a lookup table.

Parameters:

  • add_01_wrapper bool Return wrapper function?
  • lut array The underlying lookup table. If absent, a table is supplied.

    In a well-formed, populated table, each element will have number members s and t. In the first element, both values will be 0. In the final elemnt, t will be 1. In element lut[i + 1], s and t must each be larger than the respective members in element lut[i].

Returns:

  1. function Lookup function, called as

    t1, t2, index, u, s1, s2 = func(s, start)

    where s is the arc length to search and start is an (optional) index where the search may be started (when performing multiple "nearby" lookups, this might speed up search).

    t1 and t2 are the t parameter bounds of the interval, s1 and s2 are the arc length bounds of the same, index is the interval index (e.g. for passing again as start), and u is an interpolation factor ∈ [0, 1], which may be used to approximate t, given t1 and t2.

    The arc length is clamped to [0, s), s being the final s in the lookup table.

  2. array lut.
  3. optional function If add_01_wrapper is true, this is a function that behaves like the lookup function, except the input range is scaled to [0, 1].
SetLUT_Bezier2 (lut, p1, q, p2[, nsamples])
Populates an arc → parameter lookup table given a (degree 2) Bézier spline.

Parameters:

  • lut array Lookup table, cf. Lookup.
  • p1 Vector Endpoint #1...
  • q Vector ...control point...
  • p2 Vector ...and endpoint #2.
  • nsamples int Number of samples to load into lut. If absent, a default is used.

Returns:

    number Total arc length.
SetLUT_Bezier3 (lut, p1, q1, q2, p2[, nsamples[, tolerance]])
Populates an arc → parameter lookup table given a (degree 3) Bézier spline.

Parameters:

  • lut array Lookup table, cf. Lookup.
  • p1 Vector Endpoint #1...
  • q1 Vector ...control point #1...
  • q2 Vector ...control point #2...
  • p2 Vector ...and endpoint #2.
  • nsamples int Number of samples to load into lut. If absent, a default is used.
  • tolerance number "Close enough" tolerance, cf. spline_ops.bezier.Length3. If absent, a default is used.

Returns:

    number Total arc length.
SetLUT_Func (lut, how, func[, nsamples[, tolerance]])
Populates an arc → parameter lookup table given a function to integrate over [0, 1].

Parameters:

  • lut array Lookup table, cf. Lookup.
  • how string If this is "gauss_legendre", number_ops.integrators.GaussLegendre is used as the integration method. Otherwise, number_ops.integrators.Romberg is used.
  • func callable Function to integrate, e.g. an integrand supplied by spline_ops.cubic.LineIntegrand.
  • nsamples int Number of samples to load into lut. If absent, a default is used.
  • tolerance number Tolerance, as used by some integrators. If absent, a default is used.

Returns:

    number Total arc length.
