CollectArgsInto (acc, ...)
Collects arguments, including nils, into an object.

Parameters:

  • acc Accumulator object; if false, a table is supplied.
  • ... Arguments to collect.

Returns:

  1. uint Argument count.
  2. Filled accumulator.
CollectArgsInto_IfAny (acc, ...)
Variant of CollectArgsInto that is a no-op when given no arguments.

Parameters:

  • acc Accumulator object; if false and there are arguments, a table is supplied.
  • ... Arguments to collect.

Returns:

  1. uint Argument count.
  2. Filled accumulator, or acc if no arguments were supplied.
UnpackAndWipe (array, count, wipe)
Wipes an array, returning the overwritten values.

Parameters:

  • array Array to wipe. May be nil, though count must then be 0.
  • count uint Size of array; by default, #array.
  • wipe Value used to wipe over entries.

Returns:

    Array values (number of return values = count).
UnpackAndWipeRange (array, first, last, wipe)
Wipes a range in an array, returning the overwritten values.

Parameters:

  • array Array to wipe. May be nil, though last must resolve to 0.
  • first uint Index of first entry; by default, 1.
  • last uint Index of last entry; by default, #array.
  • wipe Value used to wipe over entries.

Returns:

    Array values (number of return values = last - first + 1).
WipeRange (array, first, last, wipe)
Wipes a range in an array.

Parameters:

  • array Array to wipe. May be nil, though last must resolve to 0.
  • first uint Index of first entry; by default, 1.
  • last uint Index of last entry; by default, #array.
  • wipe Value used to wipe over entries.

Returns:

    array.
PackOrGet (t, key, ...)
Resolves its input as a single result, packing it into a table when there are multiple arguments, useful when input may vary between single and multiple values.

Parameters:

  • t table Table to store multiple arguments; if nil, a fresh table is supplied.
  • key Key under which to store the argument count in the table.
  • ... Arguments.

Returns:

    Given zero arguments, returns nil. Given one, returns the value. Otherwise, returns the storage table.

See also:

    UnpackOrGet
UnpackOrGet (var, key, how)
Companion to PackOrGet, offering some options on its result.

Parameters:

  • var Variable as returned by UnpackOrGet .
  • key If var is a table with this key, it is interpreted as a packed argument table, and the value as its argument count.
  • how string Operation to request on var.

Returns:

    Returns as follows, in order of priority:

    • var is not a packed argument table: Returns var, or 1 if how is "count".
    • how is "count": Returns the argument count.
    • how is "first": Returns the first packed argument.
    • how is "rest": Returns all packed arguments after the first.
    • Otherwise: Returns all packed arguments.
