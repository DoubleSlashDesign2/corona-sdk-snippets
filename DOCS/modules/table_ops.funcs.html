Deep-copies a table.

Find (t, value, is_array)
Finds a match for a value in the table. The "eq" metamethod is respected by the search.

Parameters:

  • t table Table to search.
  • value Value to find.
  • is_array bool Search only the array part (up to a nil, in order)?

Returns:

    Key belonging to a match, or nil if the value was not found.
Find_N (t, value, n)
Array variant of Find, which searches each entry up to the first nil, quitting if the index exceeds n.

Parameters:

  • t table Table to search.
  • value Value to find.
  • n uint Limiting size.

Returns:

    uint Index of first match, or nil if the value was not found in the range.
FindNot (t, value_not, is_array)
Finds a non-match for a value in the table. The "eq" metamethod is respected by the search.

Parameters:

  • t table Table to search.
  • value_not Value to reject.
  • is_array bool Search only the array part (up to a nil, in order)?

Returns:

    Key belonging to a non-match, or nil if only matches were found.

See also:

    Find
ForEach (t, func, is_array, arg)
Performs an action on each item of the table.

Parameters:

  • t table Table to iterate.
  • func callable Visitor function, called as

    func(v, arg)

    where v is the current value and arg is the parameter. If the return value is not nil, iteration is interrupted and quits.

  • is_array bool Traverse only the array part (up to a nil, in order)?
  • arg Argument to func.

Returns:

    Interruption result, or nil if the iteration completed.
ForEachKV (t, func, is_array, arg)
Key-value variant of ForEach.

Parameters:

  • t table Table to iterate.
  • func callable Visitor function, called as

    func(k, v, arg)

    where k is the current key, v is the current value, and arg is the parameter. If the return value is not nil, iteration is interrupted and quits.

  • is_array bool Traverse only the array part (up to a nil, in order)?
  • arg Argument to func.

Returns:

    Interruption result, or nil if the iteration completed.
Invert (t)
Builds a table's inverse, i.e. a table with the original keys as values and vice versa.

Where the same value maps to many keys, no guarantee is provided about which key becomes the new value.

Parameters:

  • t table Table to invert.

Returns:

    table Inverse table.
MakeSet (t)
Makes a set, i.e. a table where each element has value true. For each value in t, an element is added to the set, with the value instead as the key.

Parameters:

  • t table Key array.

Returns:

    table Set constructed from array.
Weak (choice)
Builds a new weak table.

The table's metatable is fixed.

Parameters:

  • choice string Weak option, which is one of "k", "v", or "kv", and will assign that behavior to the __mode key of the table's metatable.

Returns:

    table Table.
