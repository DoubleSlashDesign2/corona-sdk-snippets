CleanUp (from, count)
Visits a range of objects, performing cleanup on any that have been removed, e.g. by object:removeSelf().

Cleanup of an object consists in breaking any links it has made, invalidating its proxy, and removing it from its tag's list.

Parameters:

  • from uint ID of first (possible) object.
  • count uint Number of objects to check.

Returns:

    uint First ID after visited objects.

Link:IsValid ()
Checks link validity. Links are invalid after Link:Break, or if one or both of the proxied objects has been removed by e.g. object:removeSelf().

Returns:

    boolean The link is still intact?
Link:SetBreakFunc (func)
Sets logic to call when a link becomes invalid, cf. Link:IsValid.

Called as

func(link, object1, object2, sub1, sub2)

where object1 and object2 were the linked objects and sub1 and sub2 were their respective sublinks. In the case that object* has been removed, it will be nil.

N.B. This may be triggered lazily, i.e. outside of Link:Break, either via one of the other link methods or CleanUp.

Parameters:

  • func callable Function to assign, or nil to disable the logic.
