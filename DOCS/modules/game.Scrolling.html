Functions

Follow (object, group)
Follows an object, whose motion will cause parts of the scene to be scrolled.

Parameters:

  • object DisplayObject Object to follow, or nil to disable scrolling (in which case, group is ignored).
  • group Given a value of "keep", no change is made. Otherwise, display group to be scrolled; if nil, uses object's parent.

Returns:

    DisplayObject Object that was being followed, or nil if absent.
SetRangeX (left, right)
Defines the left and right screen extents; while the followed object is between these, no horizontal scrolling will occur. If the object moves outside of them, the associated group will be scrolled in an attempt to put it back inside.

Both values should be ∈ (0, 1), as fractions of screen width.

Parameters:

  • left number Left extent = left * Screen Width.
  • right number Right extent = right * Screen Width.

See also:

    Follow
SetRangeY (top, bottom)
Defines the top and bottom screen extents; while the followed object is between these, no vertical scrolling will occur. If the object moves outside of them, the associated group will be scrolled in an attempt to put it back inside.

Both values should be ∈ (0, 1), as fractions of screen height.

Parameters:

  • top number Top extent = top * Screen Height.
  • bottom number Bottom extent = bottom * Screen Height.

See also:

    Follow
