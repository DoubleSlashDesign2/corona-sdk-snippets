Module utils.Integrators

Various numerical integrators.

For purposes of this module, an instance of type Vector is a value, e.g. a table, that has number members x and y.

BezierLength2 (p1, q, p2)
Computes a (degree 2) Bézier spline's length.

Parameters:

  • p1 Vector Endpoint #1 of control polygon...
  • q Vector ...interior control point...
  • p2 Vector ...and endpoint #2.

Returns:

    number Approximate arc length.
BezierLength2_Array (bezier)
Array variant of BezierLength2.

Parameters:

  • bezier array Elements 1, 2, 3 are interpreted as arguments p1, q, p2 from BezierLength2.

Returns:

    number Approximate arc length.

Computes a (degree 3) Bézier spline's length.

Parameters:

  • p1 Vector Endpoint #1 of control polygon...
  • q1 Vector ...interior control point #1...
  • q2 Vector ...interior control point #2...
  • p2 Vector ...and endpoint #2.
  • tolerance number "Close enough" separation between arc estimates and chord lengths.

Returns:

Parameters:

  • bezier array Elements 1, 2, 3, 4 are interpreted as arguments p1, q1, q2, p2 from BezierLength3.
  • tolerance number "Close enough" separation between arc estimates and chord lengths.

GaussLegendre (func, a, b, tolerance)
Integrates func via Gauss-Legendre quadrature.

Parameters:

  • func callable Called as

    y = func(x)

    where x and y are numbers.

  • a number Lower limit of integration.
  • b number Upper limit of integration.
  • tolerance number "Close enough" separation between quadrature estimates.

Returns:

    number Approximate integral.
LineIntegrand_Cubic (poly)
Line integrand for a cubic polynomial.

Parameters:

  • poly array The underlying polynomial, (dx/dt)² + (dy/dt)²: elements 1 to 5 are the x⁴, x³, x², x, and constant coefficients, respectively. If absent, a table is supplied.

Returns:

  1. function Integrand function, which may be passed e.g. as the func argument to the various integrators.
  2. array poly.

See also:

    SetPolyFromCoeffs_Cubic

SetPolyFromCoeffs_Cubic (poly, ax, ay, bx, by, cx, cy)
Assigns integrand coefficents (in particular, as expected by LineIntegrand_Cubic's integrand function), given a cubic polynomial's derivatives: dx/dt = 3Ax² + 2Bx + C, dy/dt = 3Dy² + 2Ey + F.

Parameters:

  • poly array Polynomial.
  • ax number Ax².
  • ay number Dy².
  • bx number Bx.
  • by number Ey.
  • cx number C.
  • cy number F.
