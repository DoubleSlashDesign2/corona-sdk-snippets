CheckboxWithText (group, x, y, text, options)
Creates a checkbox with some attached text.

Parameters:

  • group DisplayGroup Group to which checkbox will be inserted.
  • x number Checkbox x-coordinate...
  • y number ...and y-coordinate.
  • text string Text string.
  • options table

    Optional checkbox options. The following fields are recognized:

    • func: Passed as func to ui.Checkbox.Checkbox.
    • is_checked: If true, the checkbox will start out checked.

Returns:

    DisplayGroup Augmented ui.Checkbox object, with child #3: text object.
EditableString (group, keys, x, y, options)
Creates an editable text object.

Parameters:

  • group DisplayGroup Group to which text and button will be inserted.
  • keys DisplayObject ui.Keyboard object, used to edit the text.
  • x number Button x-coordinate... (Text will follow.)
  • y number ...and y-coordinate.
  • options table

    Optional string options. The following fields are recognized:

    • font: Text font; if absent, uses a default.
    • size: Text size; if absent, uses a default.
    • text: Initial text string; if absent, empty.
    • is_modal: If true, the keyboard will block other input.

Returns:

  1. DisplayObject The text object...
  2. DisplayObject ...and the button widget.

    CONSIDER: There may be better ways, e.g. put the text in the button, etc.

WallInRect (group, x, y, w, h)
Surrounds a rectangle with "walls". For various use cases, this is a viable alternative to masks for obscuring rendering and touch events outside of the rectangle.

The construct is useful so long as the "walled-in" objects remain below the walls in the display hierarchy.

Parameters:

  • group DisplayGroup Group to which wall elements are added.
  • x number Upper-left x-coordinate of rectangle...
  • y number ...and y-coordinate.
  • w number Rectangle width...
  • h number ...and height.
