GoToScene (args)
Utility that augments storyboard.gotoScene.

Parameters:

  • args table Arguments to forward to storyboard.gotoScene.

    The name field contains the scene name.

    If there is a true no_effect field, no transition effect is played. Otherwise, the array part of args may be populated with transition names; if so, one is randomly chosen as the effect. If neither of those is the case, the effect key is used.

    The params and time keys are the same as for storyboard.gotoScene.

SetListenFunc_GoBack (name, effect)
Variant of SetListenFunc for the common case of only handling "go back to the previous scene" behavior.

Parameters:

  • name string Name of return scene; if absent, uses storyboard.getPrevious.
  • effect string Effect to play on going back; if absent, uses "fade".
