Module utils.CubicSpline

Various cubic splines and related functionality.

In several of the routines, the spline type will be one of "bezier", "catmull_rom", or "hermite".

For purposes of this module, an instance of type Coeffs is a value, e.g. a table, that has and / or receives number members a, b, c, d, whereas for a Vector the members are x and y.

BezierToHermite (src1, src2, src3, src4, dst1, dst2, dst3, dst4)
Converts coefficients from Bzier (P1, Q1, Q2, P2) to Hermite (P1, P2, T1, T2) form.

This (and the similar functions in this module) are written so that output coefficients may safely overwrite the inputs, i.e. if src* = dst*.

Parameters:

  • src1 Vector Vector #1 (i.e. P1)...
  • src2 Vector ...#2 (Q1)...
  • src3 Vector ...#3 (Q2)...
  • src4 Vector ...and #4 (P2).
  • dst1 optional Vector Target vector #1 (i.e. will receive P1); if absent, src1.
  • dst2 optional Vector ...#2 (P2), or src2...
  • dst3 optional Vector ...#3 (T1), or src3...
  • dst4 optional Vector ...and #4 (T2), or src4.

GetPolyCoeffs (stype, a, b, c, d)
Gets polynomial coefficients for calculating line integrands.

A given cubic spline can be written as a polynomial Ax^3 + Bx^2 + Cx + D, for both its x- and y-components. Furthermore, its derivative is useful for computing the arc length of the spline.

Parameters:

  • stype string Spline type.
  • a Vector Vector #1 defining the spline...
  • b Vector ...#2...
  • c Vector ...#3...
  • d Vector ...and #4.

Returns:

  1. number A, for x...
  2. number ...and y.
  3. number B, for x...
  4. number ...and y.
  5. number C, for x...
  6. number ...and y.

See also:

  • utils.Integrators.LineIntegrand_Cubic
  • utils.Integrators.SetPolyFromCoeffs_Cubic
