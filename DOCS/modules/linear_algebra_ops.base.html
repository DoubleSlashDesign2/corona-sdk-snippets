EvaluateLU_Compact (out, L, U, b, n)
Evaluates the system of linear equations LUx = b, where L and U are the lower and upper triangular factors of some matrix.

Parameters:

  • out array Solution vector, i.e. x.
  • L array Lower-triangular part of factorization, stored as { c11, c21, c22, ..., cn1, ..., cnn }, where each cij is the element belonging to row i and column j.
  • U array Upper-triangular part, stored as { c11, ... c1n, c22, ..., c2n, ..., cmm, cmn, cnn }, for cij as per L and m = n - 1.
  • b array Right-hand side vector.
  • n uint Matrix dimension and number of elements in out and b.
