Interpolator:__call (step)
Metamethod.

Updates the interpolation. If it finishes, the interpolator goes into suspended mode.

Parameters:

  • step number Time step.
Interpolator:GetMode ()
Gets the current interpolation mode.

Note that the times mentioned for the "do once" options are reversed if the time is decreasing.

Returns:

    string

    Current interpolation mode, which will be one of the following:

    • "once": interpolates from time 0 to 1 and finishes.
    • "oscillate": ping-pongs between time 0 and 1 continuously.
    • "oscillate_once": interpolates once from 0 to 1 and back.
    • "suspended": time stays fixed.

Gets the current interpolation state.

  2. boolean Time is decreasing?

Interpolator:SetMap (map)
Sets the time mapping, to be applied during interpolation.

When no map is assigned, the raw time and interpolation time are the same.

Parameters:

  • map callable Map to assign, or nil to remove any mapping.

    A valid mapping function has signature

    map(t, is_decreasing),

    where t is the raw interpolation time ∈ [0, 1], and is_decreasing is true if the time is decreasing. This function must return a new time, also ∈ [0, 1].

Interpolator:Start (mode, how)
Starts an interpolation.

The interpolator must have first been assigned a duration.

Parameters:

  • mode string Interpolation mode, as per Interpolator:GetMode .
  • how string Resume command.

    If this is nil, the interpolator is reset, i.e. the interpolation time is set to 0 and increasing.

    If this is "flip", the flow of time is reversed, preserving the current time.

    If this is "forward" or "reverse", the flow of time is set as increasing or decreasing, respectively, preserving the current time.

    Finally, if this is "continue", the interpolation proceeds as it was when it was created or stopped.

See also:

  • Interpolator:SetDuration

Interpolator:__cons (interp, duration, target1, target2, context)
Class constructor.

An interpolator begins in suspended mode at time = 0 and with no mapping.

Parameters:

  • interp callable Interpolate function, which will perform some action, given the current time. A valid interpolation function has signature

    interp(t, target1, target2, context),

    where t is the current (mapped) interpolation time, ∈ [0, 1], and the remaining parameters will take whatever has been assigned as the current targets and context.

  • duration number Duration to interpolate from t = 0 to t = 1 (or vice versa), or nil to forgo setting it.
  • target1 Optional user-defined interpolation target.
  • target2 Optional user-defined interpolation target.
  • context Optional user-defined context.
