Module SparseArray

This class provides an array from which elements may be removed, without upsetting the positions of other elements and maintaining the ability to iterate the elements.

  • index uint Slot index.

Returns:

    If the slot is in use, element in the slot; otherwise, nil.

SparseArray:GetArray (null)
Gets the elements as an array.

Parameters:

  • null Value to mark unused slots.

    If this is callable, the value is instead the result of null(element), element being whatever occupies the slot.

Returns:

    array Copy of the sparse array's elements.

    N.B. The array can contain holes, either because the element itself was nil or null or any element is nil.
